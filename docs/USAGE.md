# 使用说明

## 1. 安装扩展

1. 执行 `npm install`。
2. 打开 Chrome 的扩展管理页：`chrome://extensions`。
3. 启用开发者模式。
4. 点击“加载已解压的扩展程序”，选择项目根目录或 `dist/extension`。

## 2. 提取公众号内容

1. 点击浏览器工具栏扩展图标。
2. 系统会打开独立分发页面。
3. 在“公众号文章链接”输入框粘贴 `https://mp.weixin.qq.com/...` 链接。
4. 点击“提取正文”。

提取后可查看：

- 标题
- 封面图
- 正文 HTML
- 字数、图片数、段落数
- 完整性校验结果

## 3. 内容预览

页面会直接渲染提取后的正文预览，可快速检查：

- 标题是否完整（支持修改标题文本）
- 封面是否识别成功
- 正文段落和图片是否齐全

## 4. 同步发布

1. 勾选底部目标平台。
2. 点击“同步发布”。
3. 系统将依次打开各平台发布页并自动填充标题与正文。
4. 若某个平台失败，会自动保存回退草稿，并在“发布失败回退草稿”中提供复制能力。

## 5. 错误与日志

日志面板会记录：

- 提取流程（开始/成功/重试/失败）
- 发布流程（平台进度/失败原因）
- 回退草稿保存记录

可点击“清空日志”重置。

## 6. 打包发布

执行：

```bash
npm run package
```

生成：

- `dist/fbif-oneclick-publish.crx`
- `dist/fbif-oneclick-publish.zip`
