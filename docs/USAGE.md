# 使用说明（小弹窗版）

## 1. 安装

1. 执行 `npm install`
2. 打开 `chrome://extensions`
3. 开启开发者模式
4. 点击“加载已解压的扩展程序”，选择项目根目录

## 2. 使用流程（popup → panel）

1. 点击扩展图标，打开弹窗
2. 输入飞书 `App ID / App Secret`（首次建议填写）
3. 输入飞书文档链接（仅支持 `docx/wiki`）
4. 点击按钮“提取并打开同步页”
5. 进度条从 0 到 100（无文字提示）
6. 提取完成后，手动选择同步目标（不默认）
7. 选择 `FoodTalks`：点击按钮后复制代码并打开登录页（新标签）
8. 选择 `公众号`：点击按钮后启动自动同步（待登录 → 待编辑页 → 填充中 → 已完成）
9. 若公众号自动填充失败：自动复制 HTML 正文并提示手动粘贴

## 3. 发布建议

1. FoodTalks 建议保持“复制 + 人工确认发布”
2. 公众号建议登录后直接进入图文编辑页，等待插件自动填充
3. 自动填充失败时，优先使用自动回退复制内容手动补发

## 4. 常见问题

- 链接不合法：请确认是飞书 `docx/wiki` 链接
- 无法读取文档：请先将应用（机器人）加入文档并授权
- 图片未就绪：请重试提取，等待图片拉取完成
- 公众号停留“待登录”：请在 `mp.weixin.qq.com` 完成登录
- 公众号停留“待编辑页”：请进入“新建图文”编辑页
